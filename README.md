# OnDevice AI

**Powerful AI that runs entirely on your Android phone. No internet needed. No subscription. Pay once, own it forever.**

OnDevice AI is a proprietary mobile application by On Device AI Inc. that brings on-device generative AI to Android phones. All inference runs locally using Google's Gemma models via the LiteRT framework. No data is ever transmitted to cloud servers.

**Website:** [on-device.org](https://on-device.org)

> This application is built on [Google AI Edge Gallery](https://github.com/google-ai-edge/ai-edge-gallery), licensed under the Apache License, Version 2.0. See [NOTICE.txt](Android/src/app/src/main/assets/legal/NOTICE.txt) for full attribution.

## Features

- **Run Locally, Fully Offline** — all processing happens on your device
- **AI Chat** — multi-turn conversations with context compression
- **Ask Image** — upload images and ask questions about them
- **Audio Scribe** — transcribe audio to text, translate languages
- **Prompt Lab** — single-turn prompts for summarization, rewriting, code generation
- **Bring Your Own Model** — test local LiteRT `.litertlm` models
- **Performance Insights** — real-time benchmarks (TTFT, decode speed, latency)
- **Privacy First** — zero telemetry to external servers, encrypted token storage

## Requirements

- Android 12+ (API 31)
- 3GB+ RAM (recommended: 4GB+)
- ~600MB-4.9GB storage depending on model size
