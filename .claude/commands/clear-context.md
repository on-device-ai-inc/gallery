Clear Claude context for a fresh start.

## When to Use

- Starting a completely new task
- Context getting too long
- Switching between unrelated features
- After completing a major feature

## Process

```
/clear
```

Then re-read essential files:
```
Read CLAUDE.md
Read LESSONS_LEARNED.md
```

## What Gets Lost

- Previous conversation context
- Assumptions from earlier discussion
- Partial work not committed

## What Persists

- All files on disk
- Git history
- LESSONS_LEARNED.md content
